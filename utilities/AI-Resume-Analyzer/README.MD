# 🤖 AI Resume Analyzer (NLP Powered)

An intelligent resume analysis tool built using Python and NLTK that extracts skills, evaluates resumes, and provides ATS scoring with improvement suggestions.

---

## 🚀 Features

- 🧠 NLP-based text processing using NLTK
- 🎯 Skill detection from resume text
- 📊 ATS (Applicant Tracking System) score generation
- 📄 Resume strength evaluation
- 💡 Smart improvement suggestions
- 📌 Education & experience detection

---

## 🛠️ Tech Stack

- Python 🐍
- NLTK (Natural Language Toolkit)
