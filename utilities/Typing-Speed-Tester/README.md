# ⌨️ Typing Speed Tester

A beginner-friendly Python utility to test typing speed and accuracy.

## ✨ New Features
- **Dynamic Phrases**: Now fetches fresh, random quotes from online APIs (like DummyJSON and Quotable) so you never type the same thing twice!
- **Offline Mode**: Automatically falls back to built-in sentences if you're not connected to the internet.
- **Zero Dependencies**: Still uses only pure Python standard libraries!

## Run

```bash
python Typing-Speed-Tester.py
```
