## BLACKJACK21

### Blackjack21_cli.py 
original cli version 
to play 
```bash
python Blackjack21_cli.py
```

### Blackjack21_pygame.py
pygame version with gui
no external assets

- ```suits.py``` : parameteric drawing of the suit symbols
- ```Blackjack21_pygame.py``` : main game loop

prequisite: *pygame*
```bash
pip install pygame
```
to play
```bash
python Blackjack21_pygame.py
```
